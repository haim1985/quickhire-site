AI Scoring Setup

How to connect the dashboard to the Python FastAPI scoring backend.

What the dashboard expects

The dashboard sends a POST request to /evaluate-answer and expects JSON with score, strengths, improvements, and short feedback.

Required backend endpoints

  • GET /health
  • POST /evaluate-answer

Example backend base URL

Examples:

  • http://localhost:8000
  • https://your-domain.com

Example request body

{
  "week_number": 1,
  "focus": "BA Basics",
  "deliverable": "1-page product overview",
  "learner_answer": "A Business Analyst helps the company define business problems and prepare decision-ready summaries."
}
